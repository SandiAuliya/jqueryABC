JAVA COURSE

Java course : Java is a high-level programming language. Java was created by James Gosling in 1991. Java has the motto “Write once, run anywhere”. Applications created with Java can run on various operating systems. Java is a programming language that is easy for beginners to learn. Currently, Java is widely used both for creating Desktop, Mobile (Android) and Website applications.


Here we will learn Java Programming from elementary to advanced levels, accompanied by case studies. There will be a lot of material that will be covered in this course such as:
Basic Java, discusses the basics of Java programming, discusses data types, branching, looping, creating functions, parameters and others
Java Object Oriented Programming, discusses object-oriented programming in Java, discusses how to create classes, create objects, create interfaces, perform inheritance and polymorphism, error handling, how to catch errors, and so on
Java Standard Classes, discusses classes that become the standard library in Java, discusses String, Date, Calendar, and other classes
Java Generic, discusses generic programming in Java. Discusses how to create generic classes, create generic methods, and so on
Java Collection, discusses data structures and collections in Java. Create List, Set, Map, Queue, Stack, and other data structures
Apache Maven, discusses the most popular build tools in Java. How to create a project using Apache Maven, how to download dependencies, multi module projects and how to create a Java file distribution using Maven
Java Unit Test, learn to make unit tests using Java and JUnit 5. Make mocks using the mockito framework, and more
Java Stream, discusses the Stream feature in Java, the difference with Java Collection. How to use operations on Java Stream, and more
Java Internalization, discusses how to create Java applications that support multiple languages, multiple number and date formats


Target Participants : To take part in this course, participants do not have to have prior knowledge of coding because the discussion in this course starts from the basics so that anyone who is new to the world of coding can follow it.

Learning websites starts with Web Design. Let's take a web design course at the ABC Learning Center. In this course, you will learn how to create a web page. You will learn basic knowledge about websites, from HTML, CSS to Javascript. Then you will also learn how to make responsive website pages using Bootstrap.


This course is the most comprehensive course covering all five programming: HTML, CSS, Javascript, PHP, and MySQL using Indonesian on Udemy. It doesn't matter whether you have ever had a website or have never done programming before, this course is definitely for you. Each language or program script has specific functions, namely:
HTML: to design the structure of website pages. The entire website is built from HTML structure.
CSS: to set how the website will look on the monitor screen. Whether or not the appearance of the website is set using CSS.
Javscript: to create interactive websites. Javascript is a client-side based programming script so it is often used to quickly manage interactions between users and websites.
PHP: to make the website more powerful because it can handle logic and database storage. Most of the websites in this world are built by PHP.
MySQL: to store data entered by the user into the database system.
